# 데모 시나리오

## 시나리오 1: 보험 업무 자동화 스타터

선택 항목:

- PC 웹
- 모바일 웹
- 고객앱
- PC 설치형 앱
- 고객관리
- 청구 파일 접수
- 개인 메시지
- 손사인
- PDF 자동화
- R2 파일관리

결과:

- 보험 FC/GA 업무 자동화 프로젝트 시작 문서 생성
- 권한 구조 초안 생성
- 기능 적용 지시문 생성
- 파일관리/고객앱/문서자동화 우선 개발 순서 생성

## 시나리오 2: 부동산 계약 관리 스타터

선택 항목:

- PC 웹
- 모바일 웹
- 고객용 모바일 앱
- 매물 관리
- 고객 파일 제출
- 계약서 관리
- 손사인
- PDF 좌표 자동화
- 개인 메시지

결과:

- 부동산 계약 관리 SaaS 스타터 생성
- 매물/고객/계약/파일 구조 초안 생성
- 손사인/PDF 자동화 기능 적용 지시문 생성

## 시나리오 3: B2B 거래 ERP 스타터

선택 항목:

- PC 웹
- 관리자 웹
- 판매자/구매자 권한
- 상품 등록
- 주문/발주
- 인보이스
- 메일 자동화
- 파일 업로드

결과:

- 판매자/구매자 ERP 구조 생성
- 주문/인보이스/메일 자동화 기능 적용 지시문 생성
- 권한별 메뉴 구조 초안 생성

## 시나리오 4: 오디션/지원자 매칭 스타터

선택 항목:

- PC 웹
- 모바일 웹
- 기획사 권한
- 지원자 권한
- 공고 등록
- 프로필 등록
- 영상 지원
- 심사/선발
- 스카웃 제안

결과:

- 공고/지원/심사/스카웃 플랫폼 구조 생성
- 미디어 업로드와 프로필 기능 정리
- 글로벌 확장 구조 초안 생성

## 시나리오 5: 데이터 수집/분석 스타터

선택 항목:

- 웹 앱
- 데이터 수집
- 기록 저장
- 통계/분석
- 랭킹

결과:

- 외부 API가 없는 환경에서 데이터 수집 구조 설계

- 데이터 수집 안정화 지시문 생성
