# 기능 포트폴리오

## 핵심 자산 요약

- 권한 매트릭스 엔진 (A)
- 예약 충돌 방지 엔진 (A)
- 상태 수명주기 관리 엔진 (A)
- 심사/분쟁/대시보드 계열 기능 (B/C)

## 포트폴리오 구성 방식

- 기능 단위로 "문제-해결-확장성"을 설명한다.
- 기술 스택보다 운영 효과(시간 단축, 오류 감소, 통제 강화)를 강조한다.
- 플랫폼별 적용 차이를 별도 설명한다.

## 기능 카드 템플릿

- 기능명
- 등급(A/B/C/D)
- 해결한 핵심 문제
- 적용 가능한 산업/도메인
- 도입 시 기대 효과
- 적용 리스크 및 전제 조건
