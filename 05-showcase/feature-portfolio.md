# 기능 자산 포트폴리오

## 보유 기술 자산 요약

이 기술자산보관소는 여러 프로젝트를 개발하면서 확보한 기능 자산을 정리한다.

## 문서/계약 자동화

- 손사인
- PDF 좌표 자동화
- 신청서 자동 작성
- 계약 단계별 타임라인
- 산출물 업로드/검수

## 고객/사용자 관리

- 고객 관리
- 고객 연결 코드
- 고객앱 링크
- 고객별 파일함
- 상담/메모 기록

## 메시지/공지

- 개인 메시지
- 전체 공지
- 개별 소식지
- 권한별 게시판
- 알림센터

## 파일/스토리지

- R2/S3 업로드
- 폴더 트리
- 용량 제한
- 관리자 용량 조정
- 삭제/정리

## 거래/ERP

- 상품 등록
- 주문/발주
- 인보이스
- 메일 자동화
- 판매자/구매자 권한 구조

## 매칭/지원

- 공고 등록
- 지원자 프로필
- 영상 기반 지원
- 심사/선발
- 스카웃 제안

## 데이터/분석

- 화면 기반 데이터 수집
- 기록 저장
- 통계/분석
- 랭킹
- 엑셀 업로드/컬럼 매칭

## 플랫폼/운영

- PC 웹
- 모바일 웹
- 모바일 앱
- PC 설치형 앱
- 자동 업데이트
- 디자인 시스템
- 권한 구조
