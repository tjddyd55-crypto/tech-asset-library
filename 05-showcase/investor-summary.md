# 투자/소개용 요약

## 한 줄 설명

여러 업무형 서비스 개발 과정에서 확보한 기능과 아키텍처를 재사용 가능한 기술 자산으로 정리한 프로젝트 스타터 기반 개발 플랫폼이다.

## 문제

새 프로젝트를 만들 때마다 다음 문제가 반복된다.

- 초기 아키텍처 설계 시간이 오래 걸림
- PC/모바일/앱 구조가 뒤섞임
- 디자인 기준이 흔들림
- 권한/저장소/API 구조가 프로젝트마다 달라짐
- 이미 만든 기능을 다시 개발함
- 리팩터링 비용이 커짐

## 해결

기술자산보관소는 반복되는 기능과 구조를 자산화한다.

- 프로젝트 시작 체크리스트
- 플랫폼별 아키텍처 규칙
- 디자인 시스템 규칙
- 권한 구조 규칙
- 재사용 기능 카탈로그
- Cursor/Codex 개발 지시문
- 기능별 표준화 문서

## 보유 기능 자산

- 전자서명
- PDF 문서 자동화
- 고객 파일 제출
- 개인 메시지
- 권한별 게시판
- 클라우드 파일관리
- 팀/일정관리
- 엑셀 데이터 매칭
- B2B 주문/인보이스
- 오디션/지원자 매칭
- 에스크로 타임라인
- 데이터 수집/분석

## 확장 가능 분야

- 보험 업무 자동화
- 부동산/자산거래
- 오디션/채용/지원 플랫폼
- B2B 거래 ERP
- 계약/검수/에스크로
- 데이터 수집/분석 서비스
- 병원/학원/프랜차이즈 업무관리

## 핵심 가치

- 새 프로젝트 개발 속도 단축
- 반복 기능 재사용
- 초기 구조 꼬임 방지
- 기능 단위 유지보수 가능
- 투자/협업/외주 개발 설명 자료로 활용 가능
