# 자산 지도

## 레이어별 분류

- `비즈니스 기능`: 도메인 규칙, 워크플로우, 상태 전이
- `플랫폼 구현`: PC 웹, 모바일 웹, 앱, 설치형 클라이언트 분기
- `기반 구조`: 권한, 감사로그, 캐시, 배치, 알림, 파일 처리
- `프레젠테이션`: 디자인 시스템, 컴포넌트 패턴, 화면 템플릿
- `커뮤니케이션`: 투자/소개 문구, 데모 시나리오

## 기능 등급 기준

- `A`: 다중 프로젝트 검증 + 의존성 분리 완료
- `B`: 단일 프로젝트 검증 + 경량 추상화 필요
- `C`: 개념 유효 + 구조 재정렬 필요
- `D`: 기록 보존 목적(참고만 가능)
