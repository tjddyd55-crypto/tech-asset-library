# 기술 자산 지도

## 1. 프로젝트 자산

- insurance: 보험 업무 자동화
- global-estate: 지역 기반 자산거래 플랫폼
- global_audition: 글로벌 오디션/지원자 매칭 플랫폼
- true-escrow: 블록체인 기반 에스크로/증명 서비스
- arkluk-erp: B2B 거래 ERP
- screengolf: 스크린골프 데이터 수집/분석
