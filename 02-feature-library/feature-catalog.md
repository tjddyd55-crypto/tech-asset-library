# 기능 자산 카탈로그

## 등급 기준

- A: 즉시 재사용 가능. 구조/권한/오류 처리 문서화 완료.
- B: 소폭 정리 후 재사용 가능. 특정 프로젝트 결합 포인트 일부 존재.
- C: 개념/흐름은 유효하나 리팩터링 필요.
- D: 참고용 아이디어 단계. 직접 재사용 금지.

## 1차 기능 후보

| 기능명 | 분류 | 원본 프로젝트 | 등급 초안 | 설명 |
|---|---|---|---|---|
| 손사인 | 문서/계약 | insurance | A | PC/모바일에서 서명 입력 후 이미지로 저장 |
| PDF 좌표 자동화 | 문서/계약 | insurance | B | PDF 위 좌표에 데이터를 자동 입력 |
| 고객 청구 파일 접수 | 고객앱 | insurance | B | 고객이 파일을 제출하고 담당자가 확인 |

| 전체/개별 소식지 | 공지 | insurance | B | 권한자가 공지/소식지를 작성하고 대상자가 확인 |
| R2/S3 파일 업로드 | 스토리지 | insurance, arkluk-erp | B | 클라우드 파일 업로드/다운로드 |
| 폴더 트리 파일관리 | 스토리지/UI | insurance | A/B | 고객/개인/팀 파일을 폴더 구조로 관리 |
| PC/모바일 분리 구조 | 아키텍처 | insurance | C | 플랫폼별 화면 분리 규칙 |
| Electron 자동 업데이트 | 운영 | insurance | B | PC 설치형 앱 자동 업데이트 구조 |
| 엑셀 업로드/컬럼 매칭 | 데이터 | insurance, arkluk-erp | B | 업종별 데이터 업로드/매핑 |
| 지역 기반 게시글 | 거래 플랫폼 | global-estate | B | 위치 기반 거래 게시글 구조 |

| 오디션 공고 등록 | 매칭 | global_audition | B | 모집 공고 등록 및 관리 |
