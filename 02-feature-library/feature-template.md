## 새 프로젝트 적용 지시문

```text
이 기능을 새 프로젝트에 적용한다.

원본 기능의 목적과 UX는 유지하되,
저장소, 권한, API 경로, DB 테이블명은 새 프로젝트 기준으로 연결한다.

기능 내부에 특정 프로젝트 고정값을 넣지 않는다.
PC/모바일 UI가 필요한 경우 화면 컴포넌트는 분리하고 공통 로직만 공유한다.
```
