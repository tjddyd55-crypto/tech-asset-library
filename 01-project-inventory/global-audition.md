# Global Audition 프로젝트 인벤토리

## 프로젝트 개요

기획사가 오디션 공고를 올리고 전 세계 지원자가 영상 기반으로 지원하는 글로벌 오디션/매칭 플랫폼이다.

## 설명

가수, 댄서, 연기자, 모델, 인플루언서 등 엔터테인먼트 분야에 특화된 구인/지원/스카웃 플랫폼이다.

## 사용자 유형

- 기획사
- 심사자
- 지원자
- 스카우터
- 관리자

## 주요 기능 후보

- 오디션 공고 등록
- 지원자 프로필
- 영상 기반 지원
- 지원서 제출
- 심사/선발
- 스카웃 제안
- 글로벌 다국어 대응
- 기획사/지원자 권한 분리
- 공고 검색
- 지원 현황 관리

## 재사용 가능한 기능 후보

| 기능 | 분류 | 등급 초안 | 비고 |
|---|---|---|---|
| 공고 등록 | 매칭/채용 | B | 채용/모집 플랫폼에 재사용 가능 |
| 지원자 프로필 | 사용자/프로필 | B | 영상/포트폴리오 구조 포함 |
| 영상 기반 지원 | 미디어 | C | 스토리지/변환/업로드 의존성 확인 필요 |
| 심사/선발 | 워크플로우 | B | 평가/승인 기능으로 확장 가능 |
| 스카웃 제안 | 매칭 | B | 양방향 매칭 구조로 활용 |
| Spring Boot + Next.js 구조 | 아키텍처 | B | Java 계열 스타터로 분리 가능 |

## 스타터킷에 반영할 구조

- 공고/지원/심사/선발은 범용 모집 플랫폼 패턴으로 분리한다.
- 프로필과 미디어 업로드는 독립 자산으로 관리한다.
- 기획사와 지원자 권한 구조는 B2B2C 서비스 권한 구조에 참고한다.
- Java/Spring 백엔드 스타터 후보로 관리한다.

## 우선 표준화할 기능 TOP 5

1. 공고 등록
2. 지원자 프로필
3. 지원/심사 워크플로우
4. 스카웃 제안
5. Spring Boot + Next.js 스타터 구조
