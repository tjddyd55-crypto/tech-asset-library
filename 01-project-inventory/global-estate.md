# Global Estate 프로젝트 인벤토리

## 프로젝트 개요

지역 기반으로 매물, 중고물품, 자동차 등을 거래하고 모임과 요청 기능을 제공하는 자산거래 플랫폼이다.

## 설명

우리나라 당근마켓과 유사한 지역 기반 거래 플랫폼을 목표로 한다.

## 사용자 유형

- 일반 사용자
- 판매자
- 구매자
- 모임 생성자
- 관리자
- 지역 관리자

## 주요 기능 후보

- 지역 기반 매물 등록
- 부동산/중고물품/자동차 등 카테고리 확장
- 거래 게시글
- 채팅/문의
- 모임 생성
- 필요한 사항 요청
- 위치 기반 탐색
- 신고/관리
- 사용자 프로필
- 찜/관심 기능

## 재사용 가능한 기능 후보

| 기능 | 분류 | 등급 초안 | 비고 |
|---|---|---|---|
| 지역 기반 게시글 | 거래 플랫폼 | B | 카테고리 확장 구조 확인 필요 |
| 매물 등록 | 거래 플랫폼 | B | 이미지/위치/가격 구조 포함 |
| 모임 생성 | 커뮤니티 | C | 범용 커뮤니티 기능으로 확장 가능 |
| 요청/문의 기능 | 고객/거래 | B | ERP/고객관리에도 재사용 가능 |
| 위치 기반 탐색 | 위치/검색 | C | 지도/위치 API 의존성 확인 필요 |
| 모노레포 구조 | 아키텍처 | B | 앱/패키지 분리 스타터 참고 |

## 스타터킷에 반영할 구조

- 지역 기반 서비스는 위치/카테고리/게시글/문의 구조를 분리한다.
- 거래 대상은 부동산, 자동차, 중고물품 등으로 확장 가능하게 설계한다.
- 채팅/문의 기능은 거래 도메인과 분리된 공통 메시지 기능으로 관리한다.
- 모노레포 구조는 앱과 패키지 분리에 참고한다.

## 우선 표준화할 기능 TOP 5

1. 지역 기반 게시글
2. 매물 등록
3. 요청/문의 기능
4. 카테고리 확장 구조
5. 모노레포 스타터 구조
