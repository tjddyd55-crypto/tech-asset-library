## 3. 권장 아키텍처

### 기본 원칙

- PC와 모바일 화면은 처음부터 분리한다.
- 공통 로직, 타입, API 클라이언트, 디자인 토큰은 shared 영역에 둔다.
- 플랫폼 전용 코드는 플랫폼별 폴더에 격리한다.
- 권한 체크는 프론트와 백엔드 양쪽에서 수행한다.
- 파일 저장소는 어댑터 방식으로 설계한다.
