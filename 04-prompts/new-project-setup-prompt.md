# New Project Setup Prompt

신규 프로젝트를 초기화하라.

## 입력

- 프로젝트명
- 대상 플랫폼(PC 웹/모바일 웹/앱/설치형)
- 핵심 도메인
- 필수 권한 역할
- 외부 연동(결제/알림/파일/인증)

## 작업 지시

1. `03-starter-kit` 문서 기준으로 초기 구조를 설계한다.
2. `01-project-inventory`와 `02-feature-library`에서 재사용 가능한 기능을 A/B 우선으로 선정한다.
3. 선정 기능마다 적용 이유와 필요한 조정 범위를 문서화한다.
4. 아직 코드 복사는 하지 말고, 적용 계획/경계/API 계약부터 확정한다.
5. 초기 체크리스트를 작성해 팀 합의 항목을 제시한다.
