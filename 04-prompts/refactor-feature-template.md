# Refactor Feature Template Prompt

재사용 난이도가 높은 기능(B/C 등급)을 자산화하기 위한 리팩터링 템플릿.

## 입력

- 대상 기능명
- 현재 등급
- 현재 문제점(결합/중복/권한/성능/테스트)
- 목표 등급

## 작업 절차

1. 현재 책임 혼재 지점을 식별한다.
2. 비즈니스 로직, 입출력, 인프라를 분리한다.
3. 권한/오류/트랜잭션 경계를 명시한다.
4. 테스트 가능한 인터페이스를 설계한다.
5. 자산 문서 템플릿 항목을 모두 채운다.

## 산출물

- 리팩터링 전/후 구조 비교
- 위험 요소와 롤백 전략
- 새 프로젝트 적용 지시문(단계별)
- 투자/소개용 가치 설명 문구
