# Apply Feature Template Prompt

기존 기능 자산을 새 프로젝트에 적용할 때 사용하는 템플릿.

## 입력

- 대상 기능명
- 기능 등급(A/B/C/D)
- 원본 프로젝트
- 대상 프로젝트 요구사항

## 출력 형식

- 기능명
- 원본 프로젝트
- 원본 코드 위치
- 기능 목적
- 재사용 가능 분야
- 필요한 프론트 구조
- 필요한 백엔드/API 구조
- 필요한 DB/스토리지 구조
- 필요한 권한 구조
- 외부 의존성
- 재사용 난이도
- 정리 필요 여부
- 새 프로젝트 적용 지시문
- 투자/소개용 설명 문구

## 수행 규칙

- 코드 복사 없이 경계와 계약부터 정의한다.
- 프로젝트 종속 구조는 제거 후 공통 규칙으로 치환한다.
- 플랫폼별 차이점을 별도 섹션으로 분리한다.
